# push_swap
Sorting algorithm for stack

## Running
Clone this repository:

```bash
git clone https://github.com/Mikel45/push_swap.git
```

Run sort algorithm:
```bash
cd push_swap
make
./push_swap [number in unsorted order]
```

Run checker:
Run sort algorithm:
```bash
cd push_swap
make bonus
./push_swap [number in unsorted order]
```
Type commands and press "ctrl + D" to check if it's sorted or not.
